# Firebase Console - Send Test Notification

## Finding Cloud Messaging in Firebase Console

The Firebase Console UI can vary. Here are different ways to find it:

---

## Method 1: Direct URL (Easiest)

**Just use this direct link:**
**https://console.firebase.google.com/project/botleji/messaging/compose**

This takes you directly to the compose message page.

---

## Method 2: Through Project Settings

1. Go to: **https://console.firebase.google.com/project/botleji**
2. Click the **⚙️ Settings** icon (top left, gear icon)
3. Click **"Project settings"**
4. Look for **"Cloud Messaging"** tab
5. Scroll down to test messaging section

---

## Method 3: Search in Firebase Console

1. Go to: **https://console.firebase.google.com/project/botleji**
2. Look for a **search bar** or **"All products"** link
3. Type: **"Cloud Messaging"** or **"FCM"**
4. Click on it

---

## Method 4: Left Sidebar Navigation

1. Go to: **https://console.firebase.google.com/project/botleji**
2. Look at the **left sidebar**
3. Try these sections:
   - **"Build"** → **"Cloud Messaging"**
   - **"Engage"** → **"Cloud Messaging"** or **"Messaging"**
   - **"Grow"** → **"Cloud Messaging"**
   - Or just scroll through all sections

---

## Method 5: Through Notification Center (Alternative)

1. Go to: **https://console.firebase.google.com/project/botleji**
2. Look for **"Notifications"** or **"Notification Center"**
3. Sometimes Cloud Messaging test is there

---

## If You See "Create Your First Campaign"

**You might be in the wrong section.** 

**Try:**
1. Go back to Firebase Console home
2. Look for different sections in left sidebar
3. Or use the direct URL: **https://console.firebase.google.com/project/botleji/messaging/compose**

---

## Alternative: Use Backend API to Test

**If you can't find the UI, test via your backend:**

1. Use your backend's notification endpoint
2. Send a test notification through your API
3. Your backend uses Firebase Admin SDK to send notifications

---
