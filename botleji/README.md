# Botleji

A Flutter application configured for both iOS and Android development.

## Getting Started

This project is a Flutter application that has been configured for cross-platform development.

### Prerequisites

- Flutter SDK (3.32.4 or higher)
- Dart SDK
- Android Studio / Xcode for platform-specific development
- Java 17+ for Android development

### Installation

1. Clone the repository
2. Navigate to the project directory:
   ```bash
   cd botleji
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```

### Running the App

#### For Android:
```bash
flutter run
```

#### For iOS:
```bash
flutter run
```

### Building the App

#### Android APK:
```bash
flutter build apk
```

#### iOS:
```bash
flutter build ios
```

## Project Structure

- `lib/` - Main Dart source code
- `android/` - Android-specific configuration
- `ios/` - iOS-specific configuration
- `pubspec.yaml` - Flutter dependencies and project configuration

## App Configuration

- **App Name**: Botleji
- **Package Name**: com.example.botleji
- **Platforms**: iOS and Android
